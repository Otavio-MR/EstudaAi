# Projeto IFPR - EstudaAi

O Sistema/Aplicativo, busca oferecer aos úsuarios as mais famosas formas de aprendizado, como apostilas, documentos, videos-aulas para que possam se aprofundar no conteúdo de inúmeras avaliações e provas, do Enem, Vestibular e Concursos Públicos.
